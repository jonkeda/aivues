# AiVues

Smart glasses guide for visually impaired people.

A static website helping visually impaired people set up and use Ray-Ban Meta smart glasses. Available in English and Dutch.

## Development

```bash
npm install
npm run dev
```

Opens a local dev server at `http://localhost:8080`.

## Build

```bash
npm run build
```

Output goes to `_site/`.

## Deploy

Automatic via GitHub Actions on push to `main` → GitHub Pages.
