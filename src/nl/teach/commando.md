# Spraakcommando's

Je hebt twee stemassistenten. <span lang="en">Hey Meta</span> bedient de bril. <span lang="en">Hey Siri</span> op iPhone, of <span lang="en">Hey Google</span> op Android, bedient de telefoon. Beide spelen via de brilspeakers.

<span lang="en">Hey Meta</span> commando's. <span lang="en">Hey Meta what do you see</span>, beschrijft de omgeving. <span lang="en">Hey Meta read this</span>, leest tekst. <span lang="en">Hey Meta what color is this</span>, herkent kleuren. <span lang="en">Hey Meta what is this</span>, herkent objecten. <span lang="en">Hey Meta call</span> en een naam, belt iemand. <span lang="en">Hey Meta send a message to</span> iemand <span lang="en">on WhatsApp</span>, stuurt een bericht. <span lang="en">Hey Meta play</span> en een nummer of afspeellijst, speelt muziek. <span lang="en">Hey Meta take a photo</span>, maakt een foto. <span lang="en">Hey Meta battery</span>, controleert het batterijniveau.

<span lang="en">Siri</span> commando's op iPhone. <span lang="en">Hey Siri navigate to</span> een adres, geeft looproute. <span lang="en">Hey Siri what time is it</span>. <span lang="en">Hey Siri what's the weather</span>. <span lang="en">Hey Siri set a timer for 10 minutes</span>. <span lang="en">Hey Siri remind me to</span> iets op een bepaald tijdstip. <span lang="en">Hey Siri read my notifications</span>.

<span lang="en">Google</span> commando's op Android. <span lang="en">Hey Google navigate to</span> een adres. <span lang="en">Hey Google what time is it</span>. <span lang="en">Hey Google what's the weather</span>. <span lang="en">Hey Google set a timer</span>. <span lang="en">Hey Google remind me</span>. <span lang="en">Hey Google read my notifications</span>.

De algemene regel is: <span lang="en">Hey Meta</span> voor alles wat de bril kan zien of doen. <span lang="en">Hey Siri</span> of <span lang="en">Hey Google</span> voor telefoonfuncties zoals navigatie, timers en agenda.

Wil je meer weten over een specifiek commando? Vraag het gewoon. Zeg: <span lang="en">Hey Meta, teach me about reading</span>. Of: <span lang="en">Hey Meta, teach me about navigation</span>.
