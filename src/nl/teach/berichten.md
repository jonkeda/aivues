# Berichten

Je kunt berichten sturen en ontvangen via de bril met je stem.

Voor WhatsApp, zeg <span lang="en">Hey Meta, send a message to</span> en dan een contactnaam, <span lang="en">on WhatsApp</span>. De AI vraagt wat je wilt zeggen. Dicteer je bericht en het wordt verstuurd.

Voor sms, gebruik je telefoonassistent. <span lang="en">Hey Siri, send an SMS to</span> een naam op iPhone. <span lang="en">Hey Google, send a text to</span> een naam op Android.

Om je berichten te horen, zeg <span lang="en">Hey Meta, read my messages</span>. Of als je Meldingen voorlezen aan hebt staan, worden inkomende berichten automatisch via de brilspeakers voorgelezen zodra ze binnenkomen.

Je kunt ook antwoorden op berichten die net voorgelezen zijn. Wanneer een WhatsApp-melding hardop wordt voorgelezen, kun je antwoorden door <span lang="en">Hey Meta, reply</span> te zeggen, en dan je bericht.

Tips. WhatsApp-berichten via Meta AI werken het best. Sms via Siri of Google werkt ook goed. Je kunt naar elk contact in het adresboek van je telefoon sturen.

Wil je proberen? Stuur een testbericht naar een vriend: <span lang="en">Hey Meta, send a message to</span> hun naam <span lang="en">on WhatsApp</span>.
